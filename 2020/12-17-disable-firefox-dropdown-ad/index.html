禁用 firefox 地址栏下拉网站推荐

发表于 2020-12-17 分类于 随笔

firefox 也流氓了，虽然肯定是谋智中国搞得鬼，但还是看得伤心。就像一个曾经的富家大小姐不得不到窑子里卖身一样。

估计 firefox 活不长了，虽然本人还在用。曾经我也想过，是不是 firefox 的用户多数都是像我这样的白嫖用户，所以它才那么艰难？后来我想明白了，这不过是历史的进程，如果某个人希望有人或有很多人通过微小的改变，来带动发展，从量变达到质变，那肯定是要失败的。如果希望能有改变，那就一定要从一开始就施以最暴烈的行动。就像中国革命一样，以几十年的战争，千万人的死亡为代价，才对社会稍微有了点促进。

想要逆转这个全球化，垄断化，奴隶制化的趋势，不以亿万人的死亡为考虑前提，基本就是在空想。

可能平时比较粗心的原因，在 firefox 地址栏下拉推荐里出现了知乎, 爱奇艺, 微博, 等几个网站，而我却一直没能注意到这不正常。今天因为注销知乎账号失败（国内网站流氓本性），对知乎看到就烦，甚至脑中闪过虐杀知乎员工的想法，不知道是不是我精神已经被搞得不正常了。总之终于注意到这个广告现象，于是希望将其禁止。

首先是 firefox 选项 > 隐私与安全 > 地址栏 这里，取消 常用网站 的复选框，这样以后就不会出现了。

但在网上搜索，别人还说，在 about:config 里，browser.urlbar.update1 要设置为 false，只是我自己搜索却搜索到的是 browser.urlbar.update2，不知这里有什么讲究，反正一并改为 false.

OK，问题解决！

问候千万遍知乎。
