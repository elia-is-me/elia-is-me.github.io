在弃用了 firefox 一段时间之后，终于发了这篇。或许这是我自己人生一个阶段的结束，谨以此篇纪念我的折腾时光。

最早使用 firefox 是在版本 3.x 的时候，那时刚有了电脑——也就是说在最初接触网络的时候，本人就是通过 firefox 来认识的网络世界了。彼时 ie6 的年岁已经不小，恶名也开始远扬，加上本人的性格的原因，自然是更加推崇更加自由，更加开放的 fx 了。后来有过短暂的一段时间，本人觉得得尝试下其它的浏览器，于是几乎把所有的浏览器都试了个遍，而 firefox 也确实是表现最好的那个。不是指 firefox 的扩展性，仅仅指浏览器性能体验，firefox 就是最好的，尽管那时号称最快的是 Opera。那时唯一的缺点似乎是启动速度比较慢。

<!-- more -->

然后是 firefox 4.x 的时代，不知道是模仿 Opera 还是别的，出现了火狐橙，标签页也置顶了。这时不管是外观也好，浏览器页面显示空间也好，都让我很满意，而且启动速度也很快了，至少不需要特意去注意的。标签页置顶似乎是参照了 Chrome 的设计。这期间是我瞎折腾的时间（尽管没在折腾火狐），也有几次像今天这样，弃用 firefox 使用其它浏览器，不过最后都不了了之——Opera 号称最快，但在我电脑上它就是不如 firefox 快，直到今天还是不如；chrome 太丑，而且不适应；ie ... 

然后就是 firefox 作死时代的开始，开始飙版本号，开始放弃自己的特色在界面上抄袭 chrome（抄谁不好），今年也在火狐吧看到似乎是开发人员之间的撕逼。慢慢的，对 firefox 不再喜欢，直到最近，真正地换掉了它。

新浏览器是 Opera，嗯，这么多年我每次叛逃都是到 Opera 的阵地上去的。只不过这次 Opera 的核没了，成了 Chrome 的核。还好界面完全没照着 Chrome 搬，依然是 Opera 的特色界面，加上 Chrome 的插件也开始变的丰富起来，Opera 可以搭顺风车。到现在为止，一切还好，Opera 可以担当大任。

说起来，firefox 一直以可扩（折）展（腾）性闻名于世，然而我从来没折腾过它。相比于那些定制自己界面的或者自己写 js 脚本的玩家，我几乎什么都没干过，也就是装一些现成的插件和脚本，而且很少。所以到了最近的时候，当我发现我用的扩展仅剩下 μBlock 和 lastPass 的时候，我觉得是时候跟 firefox 说再见了。

Opera 有 μBlock 和 lastPass 插件，所以转移过来完全没问题，而且再加上 adbyby 广告过滤软件的，页面广告问题基本解决。lastPass 据说有过密码泄露问题，不过对我来说算什么呢... 那些账号真的不怎么重要，重要的我也不会用它保存。以前贴吧还算良心的时候，我也用过不少油猴脚本，现在的贴吧... 其它那些稀奇古怪的扩展，有则有，无则无，似乎并没有哪个能让我留在 firefox 里。

说起来以前 firefox 把我抓住的原因是什么呢？稳定不卡顿的访问网站速度算一个，另一个能想起来的是广告过滤扩展 ADBlock。现在其它浏览器都有广告过滤，Chrome 和 Opera 的扩展也很多，那么 firefox 还有什么其它优点呢？或者它又开发出了什么新的杀手级的功能呢？

感谢一直以来的陪伴，不过仍然还是再见。
